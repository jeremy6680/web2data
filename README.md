# Web2Data

Blog personnel sur le développement web, la data et l'IA — tutoriels, insights et bonnes pratiques.

**URL** : [web2data.jeremymarchandeau.com](https://web2data.jeremymarchandeau.com/)

Construit avec [Astro](https://astro.build/) et [Tailwind CSS](https://tailwindcss.com/), basé sur le thème [tailwind-astro-starting-blog](https://github.com/wanoo21/tailwind-astro-starting-blog).

## Commandes

Toutes les commandes s'exécutent depuis la racine du projet :

| Commande                  | Action                                                    |
| :------------------------ | :-------------------------------------------------------- |
| `npm install`             | Installe les dépendances                                  |
| `npm run dev`             | Lance le serveur de dev sur `localhost:4321`              |
| `npm run build`           | Compile le site dans `./dist/`                            |
| `npm run preview`         | Prévisualise le build en local avant déploiement          |
| `npm run astro ...`       | Commandes CLI Astro (`astro add`, `astro check`, etc.)    |
